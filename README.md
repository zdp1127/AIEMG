# AIEMG

The molecular generator framework of the AIEMG is modelled on Mothra.(https://github.com/sekijima-lab/Mothra)


#### Train the RNN model
Run `python train_RNN/train_RNN.py` to train the RNN model. 
Pretrained model is provided in `model/model.h5`

#### Molecule generate
Run `python ligand_design/mcts_ligand.py ./template_for_data/`
